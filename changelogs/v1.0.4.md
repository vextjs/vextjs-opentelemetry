# v1.0.4

> 发布日期：2026-04-28
> 类型：PATCH

## 概述

本次补丁聚焦 `gRPC h2c` 导出日志噪声：默认不再为每一批成功导出持续打印 `Trace/Metrics/Logs export OK`，改为保留启动摘要、首次失败告警，以及失败后的首次恢复提示。

## 变更明细

### 日志策略调整

- `src/core/sdk-config.ts`
  - 新增导出状态感知日志器
  - 成功路径默认静默，不再每批次打印成功日志
  - 首次失败时输出 `warn`
  - 恢复后仅输出一次 `info`

### 测试补充

- `test/sdk-config.test.ts`
  - 覆盖初始成功静默
  - 覆盖连续失败不重复刷屏
  - 覆盖失败后首次恢复只提示一次
  - 覆盖恢复后再次失败重新告警

### 文档同步

- `README.md`
  - 补充 gRPC h2c 导出日志默认策略说明
- `CHANGELOG.md`
  - 新增 `1.0.4` 发布记录
